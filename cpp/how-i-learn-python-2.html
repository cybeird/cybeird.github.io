完全形态

一个完整的表达式如下：

  • _begin : 左端点，也是切片开始的地方(包括_begin本身)，而正负号则代表索引方式，正为正，负为负
  • _end : 右端点，是切片结束的地方(也包括_end本身)，当然两个端点都是可以省略的，代表从头或者到尾
  • _step : 步长，类比range，正负代表切取方向

删除元素

  • 使用remove (按值删除)
    删除单个元素，即首个符合条件的元素

  • 使用pop
    删除单个或者多个元素，按索引(元素所在位置)删除，且删除时会返回被删除的元素

  • 使用del
    根据索引删除，而且还可以删除指定范围中的值
