数据结构实验课の解题报告-Week5

发表于 2019-10-28 分类于 cpp
本文字数： 11k 阅读时长 ≈ 10 分钟
这次我们来做二叉树的一波应用，顺便好好写写实验报告

开局设置

  1. 运行VS，建立对话框工程，工程名为HFMTREE
  2. 删除窗体上的提示、确定按钮；将取消按钮的标题改为退出；
  3. 更改窗体字体字号，选择宋体，小四号字。
  4. 对话框属性的标题caption改为哈夫曼树编码译码
  5. 在对话框窗体上放置三个按钮，属性分别设置为
    | ID | Caption |
    | —————— | ——————- |
    | IDC_BUTTON_CREATE | 生成哈夫曼树 |
    | IDC_BUTTON_LOOK | 查看哈夫曼编码 |
    | IDC_BUTTON_CODING_DECODING | 编码译码 |

对应生成哈夫曼树按钮，插入新对话框

在新对话框中，实现输入数据，生成哈夫曼树，并显示静态三叉链表存储结构等功能。

  1. 删除窗体上的提示、“确定”按钮；将“取消”按钮的标题改为“退出”；
  2. 对话框属性设置
    ID               Caption           Font(Size)
    IDD_DLG_CREATE_TREE 生成哈夫曼树 宋体(16)
  3. 新对话框添加类
    鼠标右键点击窗体非控件区域，点选“添加类”选项，在新页面“类名: ”栏输入类名称“DLG_TREE_CREATE”，按“确定”按钮
  4. 在对话框窗体上添加控件
    放置一个“EditControl”、一个“Button”、一个“StaticText”、一个“ListControl”，属性分别设置为：
    • EditControl属性设置
      ID：
      IDC_EDIT_VAL
    • Button属性设置
      |ID | Caption|
      | ————- | ——– |
      |IDC_BUTTON_CREATE | 生成树|
    • StaticText属性设置
      Caption:
      输入格式：字母权值，每组可用空格分隔或不分隔，如：a80C30 D40 B70|
    • ListControl属性设置
      |ID | Sort | View | Alignment|
      | ——- | ——– | ———– | ———- |
      |IDC_LIST_CREATE_TREE | None | Report | Top|
  5. 控件绑定变量
    打开MFC类向导
    | 控件ID | 类型 | 成员 |
    | —— | —- | —- |
    | IDC_EDIT_VAL | CEdit | mc_edit_val |
    | IDE_LIST_CREATE_TREE | CListCtrl | mc_list_ctrl |

对应查看哈夫曼编码按钮，插入新对话框

在新对话框中，实现哈夫曼编码显示。

  1. 删除窗体上的提示、“确定”按钮；将“取消”按钮的标题改为“退出”；
  2. 对话框属性设置
    |ID | Caption | Font(Size) |
    | ———— | ———– | ———- |
    |IDD_DLG_LIST_CODE | 哈夫曼编码 | 宋体(16) |
  3. 新对话框添加类
    鼠标右键点击窗体非控件区域，点选“添加类”选项，在新页面“类名: ”栏输入类名称“DLG_LIST_CODE”，按“确定”按钮
  4. 在对话框窗体上添加控件
    放置一个“ListControl”，属性设置为：
    | ID | Sort | View | Alignment |
    | — | ——– | ——- | ——– |
    | IDC_LIST_CODE | None | Report | Top |
    控件绑定变量：
    | 控件ID | 类型 | 成员 |
    | —— | —- | —- |
    | IDC_LIST_CODE | CListCtrl | mc_list |

对应“编码译码”按钮，插入新对话框

在新对话框中，实现输入文本数据，发报：生成编码；输入编码数据，收报：生成报文。

  1. 删除窗体上的提示、“确定”按钮；将“取消”按钮的标题改为“退出”；
  2. 对话框属性设置
    | ID | Caption | Font(Size) |
    | —- | ——- | ——— |
    | IDD_DLG_MOVE | 发报、收报 | 宋体(16) |
  3. 新对话框添加类
    鼠标右键点击窗体非控件区域，点选“添加类”选项，在新页面“类名: ”栏输入类名称“DLG_MOVE”，按“确定”按钮
  4. 在对话框窗体上添加控件
    放置二个“EditControl”、二个“Button”、一个“StaticText”，属性分别设置为：
    • EditControl属性设置
      ID ：
      IDC_EDIT_TEXT
      IDC_EDIT_CODE
    • Button属性设置
      | ID | Caption |
      | —- | ——– |
      | IDC_BUTTON_OUT | 发报 |
      | IDC_BUTTON_IN | 收报 |
    • StaticText属性设置
      ID： IDC_STATIC_OUT
  5. 控件绑定变量
    打开MFC类向导
    | 控件ID | 类型 | 成员 |
    | —— | —- | —- |
    | IDC_EDIT_CODE | CString | m_code |
    | IDC_EDIT_TEXT | CString | m_text |
    | IDC_STATIC_OUT | CString | m_out |
